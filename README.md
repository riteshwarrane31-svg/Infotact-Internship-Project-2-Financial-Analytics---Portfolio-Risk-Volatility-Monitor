# Infotact-Internship-Project-2-Financial-Analytics---Portfolio-Risk-Volatility-Monitor
An end-to-end financial risk analysis project covering data cleaning, returns & volatility, Monte Carlo simulation, correlation analysis, and Tableau dashboards. Built as a job-ready portfolio project for Data Analyst / Business Analyst roles.

---

##  Project Objective

* Prepare and analyze financial time-series data
* Calculate daily returns, rolling volatility, and moving averages
* Simulate future return paths using Monte Carlo methods
* Analyze correlations via heatmaps
* Deliver insights through an interactive Tableau dashboard

---

##  Tech Stack

* **Python**: Pandas, NumPy
* **Visualization**: Tableau
* **Data**: CSV files
* **Version Control**: GitHub

---

#  WEEK 1: Data Collection & Cleaning

##  Goal

Prepare raw financial price data for analysis.

##  Steps

1. Load raw price data
2. Handle missing values
3. Standardize date formats
4. Validate numeric columns
5. Export cleaned dataset

##  Output

* `clean_prices.csv`

##  Key Concepts

* Time-series data
* Data preprocessing
* Financial data consistency

---

#  WEEK 2: Returns, Volatility & Trend Analysis

##  Goal

Measure asset performance and risk.

##  Steps

1. Calculate **Daily Returns**
2. Compute **30-Day Rolling Volatility**
3. Create **Moving Average**
4. Analyze **Return Distribution (Histogram)**

##  Output

* `rolling_volatility.csv`

##  Metrics

* Daily Return
* Rolling Volatility
* Moving Average

---

#  WEEK 3: Monte Carlo Simulation & Correlation

##  Goal

Simulate uncertainty and analyze dependency.

##  Steps

1. Estimate return mean & standard deviation
2. Generate Monte Carlo simulated returns
3. Store simulation results
4. Create **Correlation Heatmap** in Tableau

##  Output

* `monte_carlo_returns.csv`

##  Concepts

* Probabilistic modeling
* Risk simulation
* Correlation analysis

---

#  WEEK 4: Tableau Dashboard & Insights

##  Goal

Convert analysis into business insights.

##  Dashboard Components

* Price + Moving Average
* Daily Returns Trend
* Rolling Volatility
* Return Distribution Histogram
* Correlation Heatmap
* Interactive Date Filter

##  Final Deliverable

* **Financial Risk Analysis Dashboard**

---

##  Key Insights

* Correlation heatmaps show dependency patterns
